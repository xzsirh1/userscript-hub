## 维护原则

- 更新包统一由 `node scripts/create-update.js` 生成。
- 真实数据库、上传文件、更新包归档、临时审计目录不进入公开仓库。
- 文档中的域名、服务器路径和账号均使用占位符，部署时请替换为自己的环境信息。
