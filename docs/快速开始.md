# 快速开始指南

本指南帮助你在 5 分钟内快速启动项目。

---

## 前置要求

- Node.js >= 18.0.0
- npm 或 yarn
- Python 3.8+（用于编译 better-sqlite3）

---

## 本地开发

### 1. 克隆项目

```bash
git clone <repository-url>
cd 油猴
```

### 2. 安装依赖

```bash
# 后端依赖
cd server
npm install

# 前端依赖
cd ../client
npm install
```

### 3. 启动服务

**终端 1 - 启动后端：**
```bash
cd server
npm run dev
```

后端将在 http://localhost:3000 启动

**终端 2 - 启动前端：**
```bash
cd client
npm run dev
```

前端将在 http://localhost:5173 启动

### 4. 访问应用

- 前台首页：http://localhost:5173
- 管理后台：http://localhost:5173/admin
- 首次部署空数据库时没有默认账号，请访问 `/setup` 设置平台名称和管理员账号密码。

## 生产部署

### 方式一：Docker 部署（推荐）

```bash
cd docs/deployment-docker
chmod +x deploy.sh
./deploy.sh install
```

### 方式二：PM2 部署

```bash
# 1. 构建前端
cd client
npm install
npm run build

# 2. 安装后端依赖
cd ../server
npm install --production

# 3. 启动服务
pm2 start src/app.js --name "userscript-hub"
pm2 save

# 4. 配置 Nginx（参考 README.md）
```
